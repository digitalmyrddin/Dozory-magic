# Dozory-magic

Инструменты для браузерной игры [Дозоры](http://dozory.ru): калькулятор магии и userscripts для Greasemonkey/Tampermonkey.

---

## ✨ Калькулятор магии ВК

**[Открыть калькулятор](https://digitalmyrddin.github.io/Dozory-magic/)**

---

## 🔧Плагины для игры

Для установки плагинов нужно установить расширение [Greasemonkey в FireFox](https://addons.mozilla.org/firefox/addon/greasemonkey/) \ [Tampermonkey в PaleMoon](https://github.com/janekptacijarabaci/greasemonkey/releases/download/3.31.4Fork/greasemonkey-3.31.4-pm_forkBranch.xpi) или [Tampermonkey в Chrome](https://chromewebstore.google.com/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo?hl=en). 
После установки расширения - нажмите на ссылку скрипта и браузер предложит установку автоматически.

| Скрипт | Описание | Установить |
|--------|----------|------------|
| **FastTravel** | Быстрое перемещение по клику на карте. | [Установить](https://raw.githubusercontent.com/digitalmyrddin/Dozory-magic/main/userscripts/FastTravel.user.js) |

### Детали

— **FastTravel** :
Левый клик мышью по карте позволяет мгновенно перейти в локацию последним выбранным способом. Правый клик мышью - открыть родное окно выбора способа передвижения \ транспорта.
